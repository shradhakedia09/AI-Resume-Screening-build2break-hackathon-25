# Website: Code-Breakers

Check out our app: [https://codebreakers.streamlit.app/](https://codebreakers.streamlit.app/)

---

## 🚀 Features

- **Policy Agent**: Provides guidance based on company policies and answers employee queries.  
- **Resume Screening Agent**: Automatically evaluates resumes and shortlists candidates based on predefined criteria.  
- **Onboarding Agent**: Helps create personalized onboarding plans for new hires.  
- **Central Orchestrator (`main.py`)**: Connects all agents and manages the workflow.

## 🛠️ Tech Stack

- **Language**: Python  
- **Frontend**: Streamlit  
- **AI/Automation**: Custom Python agents using OPENAI API
- **Data Storage**: CSV files for storing resumes, policies, and onboarding data

## 🌐 Deployment

- Built with Streamlit, allowing easy deployment on Streamlit Community Cloud.  
- All agents are integrated through `main.py`, which orchestrates workflows dynamically.
